# 🤖 AI Interview Question Generator

## 📌 Project Overview

The AI Interview Question Generator is an AI-powered application that generates customized interview questions for internship and job roles using Google's Gemini Large Language Model (LLM).

The application accepts an internship role, job description, and difficulty level, then generates technical and behavioral interview questions to help candidates prepare effectively.

## ✨ Features

- AI-powered interview question generation
- Technical interview questions
- Behavioral interview questions
- Role-specific questions
- Difficulty selection (Easy, Medium, Hard)
- Interactive user input
- Automatic saving of generated questions

---

## ▶️ How to Run

1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Add your Gemini API key.
4. Run all notebook cells.
5. Enter:
   - Internship Role
   - Job Description
   - Difficulty Level
6. View and save the generated interview questions.
